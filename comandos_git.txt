git add
//adiciona arquivos modificados ou novos ao próximo commit.

git commit -m
//cria um novo commit com as modificações adicionadas ao índice, com uma mensagem de commit descritiva.

git push origin main
//envia as modificações locais para um repositório remoto, como GitHub, atualizando a branch remota com as novas modificações.

git tag -a -m
//é usado para criar uma tag anotada em um commit específico com uma mensagem descritiva.

 git push origin (tagname)
//é usado para enviar uma tag específica para um repositório remoto, atualizando assim a branch remota com a nova tag. O comando permite especificar o nome da tag a ser enviada usando o parâmetro (tagname).
